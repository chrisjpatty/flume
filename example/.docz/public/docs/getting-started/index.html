Getting Started

Installation

Flume can be installed from npm like so:

npm install flume

or alternatively

yarn add flume

Basic Usage

To get started, import and render the node editor like so:

import React from 'react'
import { NodeEditor } from "node-editor";
const App = () => {
  const [nodes, setNodes] = React.useState({})
  return (
    <NodeEditor
      nodes={nodes}
    />
  )
}

As you may guess, this alone will render only a blank editor grid with no nodes.
